GUIDE :
Insérer un sommet : Insère un sommet et lui attribue une valeur
Tracer un Chemin : Trace un arc/arete et attend la confirmation de l'utilisateur pour la mise a jour
Algorithmes : Montre la fonctionnalité de nombreux algorithmes de théorie des graphes.
Modifier :
  1. Supprimer : Supprime un élément de type nœud ou arc, choisi en cliquant sur ledit élément.
  2. Ajouter des Poids aux Chemins : Ajoute un poids de valeur 0 par défaut à tous les chemins existants. Vous pouvez modifier la valeur de chaque poids en cliquant sur les chiffres.
  3. Changer les Valeurs des Nœuds : Permet à l'utilisateur de changer les valeurs des nœuds.
  4. Sauvgarder : Enregistre le graphique dans la base de données.
  5. Charger : Charge le graphique depuis la base de données.
