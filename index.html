GUIDE:
Insert Graph: Inserts a node and gives it a value based on how many existant nodes.
Draw Path: Draws an edge by default and awaits confirmation by the user in order to make the changes final.
Delete: Deletes an element of type node or edge, chosen by clicking on said element.
Modify:
  1. Change Graph Type: Changes the graph's type from non-oriented to oriented, which changes the behaviour of the Draw Path button.
  2. Add Weight to Paths: Adds a weight of value 0 by default to every existant paths, transforming the graph into a network. You change the value of each weight by click on the numbers
  3. Change Node Values: Allows the user to change the nodes' values
  4. Confirm Graph Values: Confirms both the weight and the graph values.
  5. Save: Saves the graph into the database
  6. Save: Loads the graph from the database
Algorithms: Shows the functionality of many Graph Theory algorithms.
